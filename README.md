## 🌐 New Commands

| Say this | Stormy does |
|----------|--------------|
| *“Search for [query]”* | Performs web search, returns top result title & snippet |
| *“Weather”* | Current weather at your location |
| *“Weather in [city]”* | Weather in specified city |
| *“News”* | Top global headlines |
| *“Local news”* | News from South Africa (or your default country) |
| *“Sports news”* | Sports headlines |

## 🔑 Additional API Keys

- **OpenWeatherMap**: free tier, sign up at [openweathermap.org](https://openweathermap.org)
- **Google Custom Search JSON API**: create a Programmable Search Engine, get API key and engine ID.
- **GNews API**: free tier at [gnews.io](https://gnews.io)
